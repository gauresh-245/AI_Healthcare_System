# 🏥 AI Healthcare System

An intelligent healthcare platform built using **Python, Flask, Machine Learning, and NLP** that assists users in obtaining preliminary health assessments through disease prediction models and an AI-powered healthcare chatbot.

The system integrates multiple healthcare prediction modules including:

* Diabetes Disease Prediction
* Heart Disease Prediction
* Drug Recommendation System
* AI Healthcare Chatbot
* Symptom-Based Disease Analysis

⚠️ **Disclaimer:** This application is designed for educational and preliminary informational purposes only. It should not be used as a substitute for professional medical diagnosis or treatment.

---

# 🚀 Features

### 🩺 Disease Prediction

Predict potential diseases using machine learning models trained on healthcare datasets.

### ❤️ Heart Disease Prediction

Analyzes medical parameters to estimate the likelihood of heart disease.

### 🩸 Diabetes Prediction

Predicts diabetes risk using health indicators such as glucose levels, BMI, age, and other medical attributes.

### 💊 Drug Recommendation

Provides medicine-related recommendations based on disease information.

### 🤖 AI Healthcare Chatbot

Interactive chatbot that helps users understand symptoms, diseases, and healthcare information.

### 📊 Machine Learning Models

Pre-trained models are used for fast and accurate predictions.

### 🌐 Web-Based Interface

Simple and user-friendly Flask web application.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/gauresh-245/AI_Healthcare_System.git
```

```bash
cd AI_Healthcare_System
```

---

## Create Virtual Environment

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Linux / Mac

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run Application

```bash
python app.py
```

Open browser:

```text
http://127.0.0.1:5000
```
